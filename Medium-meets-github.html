How it works:

Edit any github repo as easy an intuitively a medium.com article:

Just replace the 'github.com' part of the url with 'qinab.com' 

For example, this .html file url is:

https://github.com/guerrerocarlos/blog/blob/master/Medium-meets-github.html

I just replaced it with:

https://qinab.com/guerrerocarlos/blog/blob/master/Medium-meets-github.html

And here you are in the medium-like editor,
